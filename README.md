**黑客派社区实时同步插件 For Wordpress**

在 WordPress 上的插件名为 [sync-hacpai](https://wordpress.org/plugins/sync-hacpai/)

同步您的Wordpress博客内容到黑客派社区

## Features
* [x] 博客发布博文 -> 社区发布帖子
* [x] 博客更新博文 -> 社区更新帖子
* [x] 博客发布评论 -> 社区发布回帖
* [x] 社区发布回帖 -> 博客发布评论

## 使用
1. 上传 zip 文件进行安装
2. 填写配置项
3. 在黑客派填写对应的配置项
