**黑客派社区实时同步插件 For Wordpress**

同步您的Wordpress博客内容到黑客派社区

## Features 
* [x] 博客发布博文 -> 社区发布帖子
* [x] 博客更新博文 -> 社区更新帖子
* [x] 博客发布评论 -> 社区发布回帖
* [x] 社区发布回帖 -> 博客发布评论

##  使用
1. 放置插件到 Wordpress 的插件目录;
2. 在设置面板进行设置.
